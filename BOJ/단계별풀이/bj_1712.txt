

a,b,c = map(int,input().split(" "))
if c - b > 0: 
    re = a // (c - b) #만들 때마다 발생하는 순이익 값으로 초기 비용을 나누면 넘는 순간을 알 수 있음. 
    if re >= 0: 
        print(re + 1)
    else:
        print(-1)
else: # 판매가격이 만드는 비용보다 작으면 손익 분기점은 절대 못넘으므로 -1 출력
    print(-1)
    
    
    - 손익분기점 문제(상품)
    - 설명
a: 초기 비용(초기 1번 발생), b: 만드는 비용, c: 판매 가격
